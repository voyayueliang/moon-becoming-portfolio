# 月亮作品系统工程化数据表

这份表是作品集的维护入口。以后新增作品，先进入这张表，再判断它应该出现在首页、展开区还是索引里。

## 首页准入规则

首页只放三类材料：

1. 能在 20 秒内帮助陌生读者理解“我是谁、我能做什么”的材料。
2. 能证明内容能力、信源能力、社区能力或交互叙事能力的代表案例。
3. 有明确数据、结果、角色或后续影响的作品。

不满足这三条的材料，先放进索引或内部素材库，不进入首页。

## 页面验收清单

- 首屏 20 秒能看懂我是谁、正在做什么、能交付什么。
- 首页代表作品不超过 8 个。
- 每个首页作品都有作品类型、我的角色、证明能力和链接。
- 完整链接在作品索引里，索引不承担主要叙事任务。
- 没有重复讲同一件事的区块。
- 没有明显 AI 味表达：不是而是、不只是更是、赋能、闭环、全链路等。
- 手机端标题、图片、按钮和作品卡片都能顺畅阅读。
